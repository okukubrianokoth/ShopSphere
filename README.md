## SHOPSPHERE
Shopsphere a full stack e-commerce website designed using React and Flask.

## User Actions
Create account and update account details.
Browse Products.
Add products to cart.
View past orders.

## Tech Stack
React:
      Formik for signup form
      Lucide-react for icons
      CSS for styling

Flask:
      SQLALCHEMY
      Migrate
      CORS
      BCrypt
      Faker
